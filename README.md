# marialababneh.com

Minimal Next.js + Tailwind CSS scaffold for Maria Lababneh's portfolio.

Local setup

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes
- This project is configured to use Tailwind CSS `4.0.0` and custom colors `primary: #F5F5DC` (beige) and `secondary: #FFFFFF` (white).
- Add your email and any images or social links in `pages/index.js`.
