# Image Assets

Add your images to this folder. Recommended images:

## Required Images:
1. **hero-brain.jpg** - Brain/mental health related image for hero section (1200x800px)
2. **research-lab.jpg** - Research lab or clinical setting (800x600px)
3. **community-support.jpg** - Veterans support group or community (800x600px)
4. **mental-health.jpg** - Mental health/wellness imagery (800x600px)
5. **profile-photo.jpg** - Your professional headshot (400x400px)

## Suggested Sources:
- Unsplash.com (free high-quality images)
- Pexels.com (free stock photos)
- Your own research/institution photos

## Image Optimization:
- Use .jpg for photos
- Use .png for graphics with transparency
- Compress images before adding (use tinypng.com)
- Target file size: < 200KB per image

## Current Placeholders:
The site uses placeholder gradients and colors until you add images.
